# Getting Started

2.  cd into the folder: `cd Fels-kning`
3.  Profit

# Build and Test
From the `Fels-kning` folder: 

1.  To build:
    1. `dotnet restore --verbosity=normal`
    2. `dotnet build --verbosity=normal`
2.  To Test:
    1. `dotnet test --verbosity=normal`
